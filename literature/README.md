# BoneBot literature & evidence

This folder is the pointer for the public **Literature** page
(`/literature` on the live site, linked from the footer). It exists so the
evidence behind BoneBot is easy to find both in the repo and in the app
itself — the "foundation value" the challenge rewards: a screening tool
whose every claim traces back to a published source or the NHANES-trained
model.

- **The page** — `src/app/literature/page.tsx`. Its `LITERATURE` register
  mirrors the "Source register" table in
  [`docs/EVIDENCE.md`](../docs/EVIDENCE.md) — same order, same wording. Add
  a row there first, then here, to keep them in sync.
- **Clinical evidence used at request time** — `src/lib/bone-evidence.ts`
  (`EVIDENCE_SOURCES`), the closed, clinician-reviewed citation list the LLM
  is restricted to when it explains a result.
- **Model performance** — `model/model-parameters.ts` (MAE, prediction-interval
  coverage, triage AUC), generated by the training notebook at
  `model/train_bonebot.ipynb`; see also `model/README.md`.
